Unbelievable Theming

Build beautiful modern Web apps with the powerful Dojo Toolkit and the awesome Semantic UI Framework. Joining Semantic UI, a development framework for creating beautiful, responsive layouts using human-friendly Web with Dojo, a JavaScript toolkit with an extensive list of powerful UI widgets. This page demonstrates a number of Dijit widgets work side-by-side with Semantic UI elements.

This theme is under active developmenet and altough a number of Dijit compoenents have been imeplemented, more development and testing is still required. For those who find this project useful and would like to contribute, pull requests are all welcomed.
